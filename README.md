# Agrinho_gabriely_oficial
Repositório para o concurso agrinho 2026, com o tema ""Tecnologia no agro (agrotech)""
# DroneField X Paraná
projeto desenvolvido para o concuro agrinho 2026.
# Tema
Tecnologia no agro (agrotech).
# Objetivo
Informar as pessoas sobre a importância sobre a Tecnologia no agro.
# Sobre
Um site informativo sobre a importância, beneficíos e assistência da tecnologia no campo.
# Funcionalidades
Exibição de cards informativos - Mostra imagens, títulos e textos sobre o tema

Navegação responsiva - Site funciona em qualquer dispositivo (celular, tablet, PC)

Interação com cards - Usuário pode clicar nos cards para visualizar informações

Mensagem de boas-vindas - Alerta automático ao entrar no site

Feedback visual no rodapé - Rodapé muda de cor quando o mouse passa por cima

# Estruturas
Principais Características:

Layout: Flexbox, footer fixo, cards alinhados

Responsivo: Breakpoints 768px/480px, cards adaptáveis

Interatividade: Hover com elevação (-8px) e transição 0.3s

Estilo: Paleta de verdes, cards arredondados (20px), sombras

Otimizações: Reset CSS, imagens responsivas, flex-grow

Tamanho: ~2KB (minificado)
